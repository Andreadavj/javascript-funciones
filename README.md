# funciones-javascript
# 🧠 Desafío 3 - Funciones

## 📋 Descripción

Este desafío tiene como objetivo poner en práctica los conocimientos adquiridos sobre funciones en JavaScript, incluyendo:

- Transformación de funciones (declaración → expresión → arrow function).
- Uso de eventos como `click` y `keydown`.
- Manipulación del DOM con eventos y cambios de estilo dinámicos.

Cada ejercicio está separado en archivos individuales para cumplir con la estructura pedida.



---

## ✅ Requisitos cumplidos

### 1. Transformación de funciones
- [x] **`1_funcion.js`**: Función `example` transformada en **función de expresión**.
- [x] **`2_arrow.js`**: Función `suma` transformada en **arrow function de una línea**.

### 2. Interacción con el DOM (`pintar.html` + `script.js`)
- [x] Script separado en `assets/js/script.js`.
- [x] La función `pintar()` recibe el elemento como parámetro para no seleccionarlo dentro de la función.
- [x] Se permite cambiar el color con argumento, con valor por defecto (`green`) y personalizado (`yellow`).

### 3. Colores y eventos (`4_colores.html`)
- [x] Cuatro `divs` con colores iniciales (azul, rojo, verde, amarillo).
- [x] Al hacer clic, el color cambia a negro con `addEventListener`.

### 4. Eventos del teclado (`keydown`)
- [x] Teclas `a`, `s`, `d` cambian el color del `div#key` a rosado, naranjo o celeste.
- [x] Teclas `q`, `w`, `e` crean nuevos `divs` con colores morado, gris y café.

---

## 🎨 Funcionalidades destacadas

- Funciones modificadas a distintas formas (declaración, expresión, arrow).
- Eventos `click` y `keydown`.
- Creación dinámica de elementos en el DOM.
- Parámetros y valores por defecto en funciones.
- Separación de responsabilidades en HTML y JS.
